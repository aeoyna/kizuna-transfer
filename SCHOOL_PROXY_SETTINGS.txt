学校（金沢工業大学）のネットワークでGitとnpmを使うための設定です。
学校に戻った時に、以下のコマンドをターミナルで実行して設定を戻してください。

=== Gitの設定 ===
# HTTPプロキシ
git config --global http.proxy http://wwwproxy.kanazawa-it.ac.jp:8080
# HTTPSプロキシ
git config --global https.proxy http://wwwproxy.kanazawa-it.ac.jp:8080

# 確認
git config --global --list

=== npmの設定 ===
# プロキシ設定
npm config set proxy http://wwwproxy.kanazawa-it.ac.jp:8080
npm config set https-proxy http://wwwproxy.kanazawa-it.ac.jp:8080

# 確認
npm config list
