# Kizuna Transfer

**「あなたのPCを、世界で最も低燃費なサーバーに。」**

広告を一切排除し、P2P技術をベースに「テラ級」のデータを安全・確実に送るプロ仕様のインフラ。

## 主要機能

### 自分PCサーバー（P2P）
*   中央サーバーを介さず、送り手と受け手のPCを直結。
*   サーバーにデータを残さないため、究極のプライバシーとコスト削減を実現。

### 「ゆとり」の予約・自動受け渡し
*   **予約送信**: 指定した時間（例：22時）に自動で通信を開始。
*   **置き配モード**: 受信者がリンクを開いた瞬間に、送信側のPCが自動でデータを送り出す。
*   「今すぐ送受信しなければ」という時間的拘束からユーザーを解放。

### テラ級対応の技術スタック
*   **超多重並列転送**: WebRTCの道を数十本束ねて回線限界まで速度を出す。
*   **完全自動レジューム**: PC再起動後も、FileSystem APIで「続き」から自動再開。
*   **低燃費設計**: WebAssemblyによる高速計算と、アイドル時のみ動くCPU制御。

## 技術構成
*   **プラットフォーム**: Vercel + GitHub
*   **フロントエンド**: Next.js (App Router)
*   **通信エンジン**: PeerJS (WebRTC)
*   **データベース**: Supabase / Firebase
*   **常駐化**: PWA
